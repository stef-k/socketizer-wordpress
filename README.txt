== Description ==

Still waiting your visitors to refresh their browser?

Quick and easy way to give life to your WordPress site

Engage your visitors with realtime posts and instant discussions,  no more F5 and refresh button spamming.

Increase your conversion rate

Transform your WordPress blog - forum - eshop into a realtime website. Give your site visitors the feel of a live
updating site;a place where they can read and write in realtime.

View the video: https://www.youtube.com/watch?v=L5XTvekMVYo

Currently Socketizer provides realtime features for the following:

 * Blog posts
 * Blog comments
 * bbPress topics
 * bbPress replies
 * WooCommerce stock change events

== Installation ==

1. Upload `com-socketizer.php` to the `/wp-content/plugins/` directory or install it from WordPress plugins directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit https://socketizer.com create an account and get your API key
4. Paste your API key in plugin's API key field located at plugin's settings
5. Enjoy your realtime site

== Frequently Asked Questions ==

= Is it compatible with my X theme - plugin? =

Yes!

= Do I need to edit X template, CSS, JavaScript, PHP code? =

No, the plugin is plug and play!

= What happens if my site reaches max concurrent visitors? =

Nothing that causes damage. The plugin works as intended for all site's visitors up to max concurrent limit.
The rest of your visitors will browse your site as normal non realtime visitors.

= What happens if my subscription ends? =

As above, your site continues to work normally as a classic WordPress site.
